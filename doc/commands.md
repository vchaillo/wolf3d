wolf3d commands
===========

### Moves

|       Key        |     Function      |
| ---------------- |:-----------------:|
| <kbd>W</kbd>     | move forward      |
| <kbd>A</kbd>     | move left         |
| <kbd>S</kbd>     | move backward     |
| <kbd>D</kbd>     | move right        |
| <kbd>SPACE</kbd> | jump              |

### Other

|        Key        |        Function        |
| ----------------- |:----------------------:|
| <kbd>ESCAPE</kbd> | exit                   |
